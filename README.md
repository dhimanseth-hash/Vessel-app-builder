# Vessel Consistency Checker

Static prototype for investor-material consistency checking.

## Deploy on Vercel

Upload this folder to GitHub with these files at the repository root:

- `index.html`
- `vercel.json`
- `README.md`

Then import the repository in Vercel.

Use these settings:

- Framework Preset: `Other`
- Root Directory: `.`
- Build Command: leave blank
- Output Directory: leave blank

The app is self-contained. The sample documents, logo, CSS, and JavaScript are embedded in `index.html`.
